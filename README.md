HPG 6104 – Epidemiological Methods I (2025-2026)Welcome to the course repository for HPG 6104: Epidemiological Methods I at the University of Nairobi. This repository houses all the interactive course materials, student handouts, and web-based calculators designed to support the 2025-2026 academic year.🌐 Access the Live Course Portal here: https://mndiritu.github.io/hpg6104_2025-2026/📖 OverviewThe goal of this project is to move beyond static PDFs and PowerPoint presentations by providing Epidemiology students with interactive, browser-based tools that reinforce core concepts. The materials focus on applied public health math, study designs, and data interpretation.🗂️ Course Content DirectoryWeek 1: Measures of Disease Frequency & AssociationCourse Portal (index.html): The main landing page for students to navigate weekly materials.Student Handout (week1_handout.html): A printable, mobile-friendly quick-reference guide covering the 2×2 table, prevalence, incidence, relative risk (RR), and odds ratios (OR).2×2 Epi Calculator (epi_calculator.html): An interactive tool where students can plug in $a, b, c, d$ values to instantly calculate and interpret risk, RR, OR, and Attributable Risk.Practice Quiz (week1_quiz.html): A self-paced, auto-grading quiz that tests students on denominator selection and epidemiological metrics.Upcoming ModulesWeek 2: Categorical Data Analysis & Contingency Tables (Chi-square, Fisher’s exact, Mantel–Haenszel)(Additional weeks will be added as the semester progresses)🛠️ Technology StackThese tools are built to be lightweight, fast, and accessible on any device (including mobile phones with poor internet connections).HTML5 & CSS3Tailwind CSS (via CDN for rapid styling)Vanilla JavaScript (for interactive calculators and quiz logic without heavy framework dependencies)🎓 For Educators & ContributorsIf you wish to reuse or modify these materials for your own courses:Clone the repository.Open index.html in your browser.No build tools or server environments are required—everything runs natively in the browser.Developed for educational use.
